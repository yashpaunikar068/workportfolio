Evaluating answers

We picked user stories to be coded in ReactJS.

  • User Story: As a student, I should be able to know my score and detailed result after completion of the assessment

My part was to complete the user story wherein my job was to check the feilds entered by the student and evaluate it according to the answers.

All the answers to the fillups were registered in a list and then they were passed as props to the result component.

We wanted the test taker to be not able to see the answers so answers were needed to be hidden. Those answers should only be retrieved from the backend only after the test is over. For retrieving the answers I wrote a dummy JSON which stored the actual answers

Some Obstacles

Faced a issue on retrieving the data form github as JSON which was later resolved with the help of teammates.
