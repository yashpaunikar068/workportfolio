Writing UI for Users

We picked up 3 user stories to be coded in ReactJS.

  • User Story 1: As a student, I should be able to click on the assessment to see the instructions, so that I know what to expect in the assessment.
  • User Story 2: As a student, I should be able to start the assessment to see the first question, so that I can read the question and answer it.
  • User Story 3: As a student, I should be able to click on the next question, so that I can see the next question or finish the assessment in case it is the last question.

My part was to complete the user story 2 wherein my job was to start a assesment.

I created a React app and implemented a basic navigation system among the questions along with one my teammates. Here I did not focus much on styling but focused on getting the functionality.
